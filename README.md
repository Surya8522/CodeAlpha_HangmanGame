# 🎮 CodeAlpha Hangman Game

## 📌 Overview
This project is a text-based Hangman game developed using Python as part of the CodeAlpha Python Programming Internship. The player guesses a randomly selected word one letter at a time within a limited number of attempts.

## ✨ Features
- Random word selection
- 6 incorrect attempts allowed
- Console-based gameplay
- Win and lose conditions
- User-friendly interface

## 🛠 Technologies Used
- Python 3
- Random Module

## ▶️ How to Run
1. Install Python 3.
2. Download or clone this repository.
3. Run:
```bash
python hangman.py
```

Enter a letter: p
