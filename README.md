## 📌 Project Overview

Hangman Game is a console-based Python application where players attempt to guess a randomly selected word by entering one letter at a time.

The game provides a limited number of incorrect attempts, challenging players to use logical thinking, vocabulary skills, and strategic guessing to discover the hidden word before the attempts run out.

---

## ✨ Features

- 🎲 Random word selection
- 🔤 Letter-by-letter guessing system
- ❤️ Limited incorrect attempts
- ✅ Input validation
- 🏆 Win and lose conditions
- 💻 Interactive console interface
- 🔄 Replay-friendly gameplay

---

## 🚀 How To Run

### Step 1: Install Python

Ensure Python 3 is installed on your system.

### Step 2: Run the Application

Open the terminal inside the project folder and execute:

```bash
python hangman.py
```

### Step 3: Play the Game

- Enter one letter at a time.
- Try to guess the hidden word.
- Avoid using all available attempts.
